## Features

- 保護耳朵，避免意外將音量調到超大聲
- 切換聲音輸出裝置後仍可維持限制

### Background

此專案的靈感來自於我在使用 Logitech K580 鍵盤時遇到的一個問題。當透過其音量按鍵調整音量時，鍵盤經常會導致音量無限制地連續重複，音量飆升到極高的程度。這對於佩戴耳機時尤為麻煩，因為我的常用音量範圍在 4 到 12 之間，而音量的突然飆升可能會對聽力造成損害。

羅技客服根本沒用，所以這個專案就誕生了
